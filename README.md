This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## About the app

This is a web app project created using NextJs and consuming Star Wars Api (SWAPI).

## Detailed Features

- List of films
- The list is searchable
- Films can be favourited/unfavourited and saved in local storage
- Each film links to a film details page
- Films bring the characters and their details using a tooltip

## To run the app

To open it locally on http://localhost:3000 using the browser.
First clone the repository and run yarn install and then yarn dev
